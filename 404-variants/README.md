# 404 Page Variants
Adapted the 404.html page into React (static), Go (server), Rust (server).

## React
Static SPA - open `index.html`.

## Go
`cd go-comparison && go mod init 404 && go mod tidy && go run main.go` - visit http://localhost:8080

## Rust
`cd rust-comparison && cargo build && cargo run` - visit http://localhost:8080 (requires Rust toolchain)
